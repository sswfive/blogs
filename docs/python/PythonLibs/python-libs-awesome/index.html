Python常用库汇总[awesome]

Web开发库​

  • Django: 一款全面且重量（大而全）的Web框架，当前star数： 75.1k
  • FastAPI: 一款高性能、异步的轻量级Web框架，当前star数：67.2k
  • Flask: 一款轻量级用于构建Web应用程序的Python微框架， 当前star数：65.5k
  • streamlit: 一款快速搭建数据共享与可视化的Web应用框架，当前star数：29.7k
  • django-rest-framework: 一款基于Django，用于快速开发Web API程序的扩展框架，当前star数：27k
  • gradio:  一款用于快速构建AI算法可视化部署的框架，当前star数：25.7k
  • Tornado: 一款异步的网络库，可用于快速开发Web应用的框架，当前star数：21.4k
  • sanic: 一款快速构建、快速开发的异步Web框架，当前star数：17.6k
  • Falcon：面向python开发人员的简单Web数据平面API和微服务框架，当前star数： 9.3k
  • nicegui: 一款使用python开发的Web用户界面，当前star数： 6.4k
  • nameko:一款用于构建微服务的Python框架，当前star数：4.6k
  • cherrypy: 一个轻量级、Pythonic、面向对象的HTTP 框架，当前star数：1.8k
  • Microdot: 适用于Python和Microdot的超小型Web框架；当前star数：1.2k

Python 机器学习库​

  • lightautoml：一款全自动机器学习框架， 当前 star 数：720

Python 数据处理库​

  • voluptuous: 一个 Python 数据验证库,当前 star 数：1.8k

ORM 库​

  • sqlalchemy: Python 数据库工具包
  • databases: Python的异步数据库支持
  • django-orm:
  • peewee
  • tortoise-orm
  • alembic: 数据库迁移工具

环境包管理库​

  • conda: 包管理工具
  • poetry: 依赖管理工具
  • pdm: 依赖管理工具
  • pipx: 管理Python包的工具,在隔离环境中安装和运行 Python 应用程序
  • pipenv: 管理Python包的工具
  • virtualenv: 虚拟环境管理工具
  • venv: 虚拟环境管理工具
  • virtualenvwrapper: 虚拟环境管理工具
  • pyenv: Python版本管理工具

任务库​

  • celery: 分布式任务调度框架
  • apscheduler: 定时任务调度框架
  • funboost: 异步任务调度框架
  • rq: 轻量级任务调度框架
  • sched: 轻量级任务调度框架
  • schedule: 轻量级任务调度框架

性能分析库​

  • cProfile: 标准库自带的性能分析
  • memory_profiler: 内存分析工具，监控Python代码的内存使用情况
  • line_profiler: 行级性能分析工具，

代码编译加密库​

  • Nuitka: 代码加密工具,一个用 Python 编写的 Python 编译器
  • pyarmor: 代码加密工具
  • py2sec: 一个跨平台、快速且灵活的工具，可将 .py 更改为 .so（Linux 和 Mac）或 .pyd（Win）
  • pyinstaller: 代码打包工具,将 Python 程序冻结（打包）为独立的可执行文件

测试库​

  • pytest: 测试框架
  • unittest: 单元测试框架
  • pytest-html: 测试报告生成工具
  • mock: 模拟对象库

依赖注入框架​

  • dependency-injector: 依赖注入框架
  • bevy: 依赖注入框架

重试库：​

  • retry: 重试库

进程管理库​

  • supervisor: 进程管理工具
  • gunicorn: 进程管理工具
  • uwsgi: 进程管理工具
